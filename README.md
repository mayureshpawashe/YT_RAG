# YouTube RAG Chatbot

A Python application to build a Retrieval-Augmented Generation (RAG) chatbot for YouTube videos. It allows you to add YouTube video transcripts to a local knowledge base and ask questions about them, either via the command line or a Gradio web UI.

---

## Features

- **Add YouTube Videos:** Add one or more YouTube videos by URL. Their transcripts are fetched and stored.
- **Chat:** Ask questions about the content of added videos, with answers generated by an LLM.
- **Web UI:** Use an interactive Gradio web interface.
- **Statistics:** View knowledge base stats, such as number of documents, unique videos, and storage info.
- **Knowledge Base Management:** Delete videos, reset the database, and more.

## Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/mayureshpawashe/YT_RAG.git
   cd YT_RAG
   ```

2. **Create a virtual environment (optional but recommended):**
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows use `venv\Scripts\activate`
   ```

3. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

## Environment Variables (`.env` file)

The application uses environment variables for configuration. Create a `.env` file in the root directory with the following contents:

```dotenv
# Required for LLM access (Groq API)
GROQ_API_KEY=your_groq_api_key_here

# Optional: Change the embedding model (default is 'all-MiniLM-L6-v2')
EMBEDDING_MODEL=all-MiniLM-L6-v2

# Optional: Local path for ChromaDB database
CHROMA_DB_PATH=./chroma_db

# Optional: Collection name for ChromaDB
COLLECTION_NAME=youtube_transcripts

# Optional: Chunk size for text processing
CHUNK_SIZE=1000

# Optional: Chunk overlap for text splits
CHUNK_OVERLAP=200
```

**Note:** At minimum, you must set `GROQ_API_KEY`.

## Usage

### Command-line

```bash
# Add a video and start chatting
python main.py --add https://www.youtube.com/watch?v=VIDEO_ID

# Add multiple videos
python main.py --add URL1 URL2 URL3

# Start chat (using existing knowledge base)
python main.py --chat

# Show statistics
python main.py --stats

# Delete a specific video
python main.py --delete VIDEO_ID

# Reset the entire knowledge base
python main.py --reset

# Launch Gradio web UI
python main.py --ui

# Launch Gradio UI with public link
python main.py --ui --share
```

### Gradio Web UI

1. Run:
   ```bash
   python main.py --ui
   ```
2. Open the local link printed in your terminal (default: http://localhost:7860).

## Project Structure

- `app.py` - Gradio web interface

- `config.py` - Loads configuration from environment and validates required settings
